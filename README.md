# Pandemic Resource Allocation under Uncertainty

This repository contains the datasets and code used in the research paper:

**"A Risk-Averse Stochastic Program for Pandemic Resource Allocation under Uncertainty"**  
by Reza Abazari and Arda Vanli.

## Overview
This study develops a risk-averse stochastic programming framework for allocating healthcare resources during pandemics under uncertainty. The model integrates epidemic dynamics with interregional capacity sharing and accounts for severe outbreak scenarios using Conditional Value-at-Risk (CVaR).

## Contents
- `data/` – datasets used in the study  
- `code/` – implementation of the models and experiments  

## Notes
The code is provided for research and reproducibility purposes.  
Please refer to the paper for full methodological details.
